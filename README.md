# Dream Engine

## Requirements

- [rust](https://www.rust-lang.org/tools/install)
- [wasm-pack](https://rustwasm.github.io/wasm-pack/installer/)

## Get Started (Desktop)

```shell
cargo run
```

## Get Started (Web)

```shell
wasm-pack build --target web --out-dir web/build
```

Serve ``web/index.html``
