## Requirements

- [rust](https://www.rust-lang.org/tools/install) (Version 1.68.0)
- [wasm-pack](https://rustwasm.github.io/wasm-pack/installer/) (Version 0.10.3)

## Get Started (Desktop)

```shell
cargo run
```

## Get Started (Web)

```shell
./tools/build-web.sh
```

Serve ``web/index.html``
